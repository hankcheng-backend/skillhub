**`init_db` function:** Update to:
- Check `is_fresh` before doing anything (same as current)
- If NOT fresh (DB exists), backup the DB file to `db_path.with_extension("db.bak")` using `std::fs::copy`. Per D-10, backup BEFORE opening the connection (pitfall 3 from research: avoids WAL sidecar corruption). If copy fails, log a warning but continue (backup is safety net, not hard requirement).
- Open connection with `let mut conn = Connection::open(db_path)?;` (must be `mut` for `to_latest`)
- Run PRAGMAs
- Run `Migrations::from_slice(MIGRATIONS).to_latest(&mut conn)` wrapped in error handling
- On migration failure (per D-11): drop the connection, restore from `.bak` if it exists, re-open the connection on the pre-migration schema, and return the error. The caller in `lib.rs` will handle showing the error dialog.
- Return `Ok((conn, is_fresh))` on success

```rust
pub fn init_db(db_path: &Path) -> Result<(Connection, bool), AppError> {
    let is_fresh = !db_path.exists();

    // D-10: Backup existing DB before migration (before opening connection)
    let bak_path = db_path.with_extension("db.bak");
    if !is_fresh {
        if let Err(e) = std::fs::copy(db_path, &bak_path) {
            log::warn!("Failed to backup DB before migration: {}", e);
        }
    }

    let mut conn = Connection::open(db_path)?;
    conn.execute_batch(
        "PRAGMA journal_mode=WAL;
         PRAGMA foreign_keys=ON;",
    )?;

    let migrations = Migrations::from_slice(MIGRATIONS);
    if let Err(e) = migrations.to_latest(&mut conn) {
        log::error!("Migration failed: {}", e);
        // D-11: Restore from backup on failure
        drop(conn);
        if bak_path.exists() {
            if let Err(restore_err) = std::fs::copy(&bak_path, db_path) {
                log::error!("Failed to restore DB from backup: {}", restore_err);
            }
        }
        // Re-open the pre-migration DB so the app can still run
        let fallback_conn = Connection::open(db_path)?;
        fallback_conn.execute_batch(
            "PRAGMA journal_mode=WAL;
             PRAGMA foreign_keys=ON;",
        )?;
        return Err(AppError::Migration(format!(
            "Database migration failed: {}. The database has been restored from backup.",
            e
        )));
    }

    Ok((conn, is_fresh))
}
```
